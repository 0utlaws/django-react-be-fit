# Django-React-BE-FIT - nutritional application




## Description:
   -  The app was created to follow users nutrition habits and help them keep desired weight. Logged in user can keep daily meal diary which is compatible with introduced caloric demand. He can add product to the database and adjust their quantity to the particular meals.
    Their calories are authomaticly ancounted. Every product has assigned information about contained fats, carbs and proteins. Thanks to the application user has control on calories and macronutients consumed during the day.

## Technologies:

  - Python
  - JavaScript
  - Django Rest Framework
  - React
  - HTML5
  - CSS3
